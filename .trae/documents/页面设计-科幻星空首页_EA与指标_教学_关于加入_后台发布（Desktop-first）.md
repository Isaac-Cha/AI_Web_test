# 页面设计与内容规范（Desktop-first）

## 全局设计（适用于全站）
- Layout：桌面优先，12列栅格（max-width 1200~1320px），内容区居中；移动端降为单列并折叠导航。
- Meta（默认）：
  - title："EA策略与指标｜合作平台展示"
  - description："展示合作平台、EA列表、指标解释与教学资源，支持多语言与后台发布。"
  - OG：og:title/og:description/og:type=website/og:image（后台可配默认图）
- Global Styles（Tokens）：
  - 背景：#070A12（深空）+ 星尘渐变
  - 主色：#6EE7FF（霓虹青）；强调：#A78BFA（紫）；危险：#FB7185
  - 字体：中文优先（如思源黑体/系统字），英文 Inter；字号：12/14/16/20/28/40
  - 按钮：Primary 霓虹描边+轻发光；Hover 增亮+阴影；Focus 可见描边
  - 链接：默认下划线弱化，Hover 强化并显示外链图标
- i18n：右上角语言切换（zh-CN/en），切换后更新 URL query（如 ?lang=en）或本地存储并立即生效。
- SEO关键词方案（首页落地）：
  - 关键词簇："EA"、"外汇EA"、"量化策略"、"回测指标"、"最大回撤"、"夏普"、"收益回撤比"、"交易教学"
  - 结构：H1承载主关键词；合作平台/EA精选/指标亮点各一段可抓取文本；FAQ区块覆盖长尾问答
  - 内容来源：由后台维护（site_settings.seo_home），避免写死在前端

## 1) 首页（/）
- Page Structure：首屏 Hero + 分区信息流（平台/EA/指标/教学/FAQ）+ 页脚。
- Sections & Components：
  1. Top Nav：Logo（左）、导航（中）、语言切换（右）、"加入/咨询"按钮（右侧高亮）。
  2. 星空互动动效层：Canvas/WebGL 全屏背景；与内容分层（z-index），默认不遮挡文本；提供“降低动效”开关（尊重 prefers-reduced-motion）。
  3. Hero：
     - H1："合作平台与EA策略指标一站式展示"
     - Subcopy：1-2行 SEO 文案（后台可配）
     - CTA："查看EA"、"查看指标"
  4. 合作平台展示：Logo卡片网格（3-4列），卡片含简介与外链。
  5. EA精选：横向卡片或3列网格；显示关键指标摘要；"进入EA列表"。
  6. 指标亮点：3~6个指标摘要卡；"进入指标列表"。
  7. 教学入口：教学分类卡（如“入门/回测/风控”）+ 最新链接。
  8. FAQ/关键词落地：可折叠问答组件（后台维护），提升长尾覆盖。
  9. Footer：版权、外链声明、联系入口。

## 2) EA 列表（/eas）
- Layout：左侧筛选栏 + 右侧列表（桌面）；移动端筛选抽屉。
- Components：
  - 筛选：标签、风险等级、发布状态（仅后台可见）、排序（更新时间/热度/指标）。
  - 列表卡片：名称、摘要、标签、3个关键指标位（占位文本由后台维护或固定字段）。
  - 详情进入：点击卡片进入详情（可用弹层或跳转锚点区块）。

## 3) 指标列表（/metrics）
- Layout：顶部分类Tab + 指标表格/卡片列表。
- Components：
  - 搜索框：按名称/代码过滤。
  - 指标条目：名称、定义、注意事项（折叠展开），相关EA/教学入口（若配置）。

## 4) 教学资源（/learn）
- Layout：两栏（左分类/右列表）。
- Components：
  - 链接卡：标题、标签、来源提示；外链打开新窗口并提示。
  - 空状态：无结果时提示调整关键词。

## 5) 关于 / 加入（/about）
- Page Structure：上方关于内容 + 下方加入表单 + 侧边联系方式。
- Components：
  - 关于内容：富文本块（后台维护，多语言）。
  - 加入表单：姓名（可选）、联系方式（必填）、意向、备注；提交成功页内提示。
  - 微信二维码浮窗：右下角悬浮按钮（全站一致）；弹窗含二维码图片、复制微信号（可选）、关闭。

## 6) 后台登录（/admin/login）与内容管理（/admin）
- 登录页：邮箱/密码、错误提示、加载态。
- 后台Layout：左侧侧边栏（平台/EA/指标/教学/站点配置），右侧编辑区。
- 编辑器：表单化字段（slug、发布状态、排序、多语言字段以 Tab 切换 zh-CN/en）；保存/发布按钮；删除需二次确认。
- 站点配置：
  - 微信二维码上传（Storage）并预览
  - SEO默认Meta、首页关键词落地文案/FAQ 编辑
  - 默认语言与回退策略
